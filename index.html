    What is RP-Commons?

    RP-Commons is a plugin development library that is currently in alpha. It's intended goal is to allow for easier creation of Bukkit Plugins, while eliminating as much of the common "boiler plate" code as possible. It allows for simple management of common plugin constructs, such as commands, permission management, configuration, and localization.

    How can this library be used?

    This library can be used for all aspects of development. It can be modified and redistributed in any way you wish, so long as you comply with the license agreement. It's completely free to use, plus it's open source! So there's no need to buy usage licenses or pay royalties. Just grab it and go!
